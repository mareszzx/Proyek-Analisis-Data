# E-Commerce Data Analysis Project

## Project Overview
Proyek ini bertujuan untuk menganalisis data E-Commerce Public Dataset guna menjawab pertanyaan bisnis terkait profitabilitas produk, perilaku belanja pelanggan berdasarkan geografis, dan distribusi demografi pelanggan di Brazil.

## Persiapan Environment
1. Clone repositori ini ke komputer Anda.
2. Pastikan Anda memiliki Python terinstal.
3. Instal semua library yang dibutuhkan:
   ```bash
   pip install -r requirements.txt

## Menjalankan Dashboard Streamlit
### Masuk ke direktori proyek (folder dashboard), lalu jalankan perintah berikut:

```bash
streamlit run dashboard.py
```

## Struktur Proyek

- /dashboard: Berisi file utama dashboard (dashboard.py) dan data yang sudah dibersihkan (main_data.csv).

- /data: Berisi raw data (opsional, jika ukurannya tidak terlalu besar).

- notebook.ipynb: File analisis data lengkap mulai dari Wrangling hingga Visualisasi.
